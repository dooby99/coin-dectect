# 제작하게된 이유
 - 많은 동전을 분류하고 총액 을 파악하는데 어려움이 있는 
   장애인이나 어린이등 뿐만아니라 우리 모두 일상에 편리함을 주기위해 제작 

# 목표
 - 동전에 적힌 숫자와 그림을 인식하여 동전 종류 파악 
   파악한 동전의 종류에 따라 사진 속 동전 종류의 각 갯수와 총 금액 합산 도출


# DATA
 - 10원, 50원, 100원, 500원 총 4가지의 앞, 뒷면,
 - 정확도 향상을 위해 기울여 촬영된 동전사진
